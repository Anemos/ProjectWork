========================================================================
Комплект файлов:
 Readme.ru.txt      - этот файл (рус)
 Readme.en.txt      - этот файл (анг)
 pbldump.exe    - исполняемый файл программы (win32 console application)
 custom.ini.add.ru  - файл с настройками для FAR'а (рус)
 custom.ini.add.en  - файл с настройками для FAR'а (анг)
 pb-colorer4.zip\powerbuilder.hrc - файл с расцветкой PowerBuilder для Colorer 4
 pb-colorer5.zip\powerbuilder.hrc - файл с расцветкой PowerBuilder для Colorer 5
 test_app.pbl    - тестовая библиотека из поставки PowerBuilder'а
========================================================================
Поддерживаются режимы:
- листинг имен, времени модификации и размеров объектов и их p-кода
    Пример:  pbldump -v test_app.pbl
    
- извлечение объектов/p-кода в файлы в текущем каталоге
    Пример:  pbldump -e test_app.pbl m_genapp_frame.srm m_genapp_frame.men
    
- совместная работа с FAR'ом (работа с PBL/PBD как с каталогами)
  см. файл "custom.ini.add.ru"

- работа с форматами библиотек PowerBuilder всех версий
  
========================================================================
Не поддерживается (и не будет) любая модификация исходной библиотеки.
См. компилятор PowerBatch (http://sqlbatch.com/pbc/) для редактирования
объектов прямо в FAR

История версий 

11.05.2009  1.3.1
    Исправлена некорректная работа с PBL защищенными от записи

23.12.2007  1.3.0
    Добавлена возможность экспортировать SCC data из PBL:
              pbldump -e test.pbl $sccdata$\sccdata

    Добавлена возможность при экспорте установить время модификации файла 
              равным времени объекта в библиотеке (ключ t):
              pbldump -et test.pbl test.sra

25.06.2004  1.2.0 
    Добавлена поддержка PowerBuilder 10 и Unicode 
    Добавлено преобразование Unicode->ANSI и ANSI->Unicode (опции "a" и "u")
    Добавлена возможность экспортировать только исходные тексты если 
              задана маска "*.*" (опция "s")
    Добавлена поддержка экспорта по файлу со списком объектов, это значительно
              ускоряет экспорт большого числа объектов в FAR:
              pbldump -e test.pbl @list.txt

13.07.2003  1.0.5
    Добавлена поддержка экспорта бинарных данных OLE объектов. Соответственно
    в PowerBatch добавлена поддержка их импорта.


23.02.2003  1.0.4
    Добавлена поддержка PBD внутри EXE (файлы до 128Кб, иначе - 
      только из командной строки )
      например: pbldump -v pbvm60.dll


29.03.2000  1.0.1 - 1.0.3
    Протестирована поддержка библиотек PowerBuilder версии 7 и выше
    Добавлен листинг времени модификации объектов для FAR'a
    Исправлены ошибки

20.03.2000  1.0
    Начальная версия
